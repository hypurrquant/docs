The **HypurrQuant** Telegram bot provides a streamlined interface for interacting with trading features powered by the Hyperliquid.

With just a few commands, users can:

- Trade both **spot** and **perpetual** markets using Hyperliquid L1.  

- The **Spot Buy Strategy** command allows you to buy Hyperliquid Core using predefined quantitative trading strategies.

- **Create**, **import** and **export** EVM wallets directly through the bot for secure platform architecture.

- Use the **balance** command to quickly check current asset balances and open positions

- **Receive P&L alerts** in real-time through customizable alarms

- Access powerful **Copy Trading** to automatically follow and replicate trades from top-performing accounts
