# Overview
HypurrQuant is a Python package designed to facilitate the analysis of high-throughput sequencing data, particularly in the context of single-cell RNA sequencing (scRNA-seq) and bulk RNA sequencing. It provides a comprehensive set of tools for data preprocessing, quality control, normalization, differential expression analysis, and visualization.
The package is built on top of popular libraries such as NumPy, pandas, and Matplotlib, ensuring compatibility with existing data analysis workflows. HypurrQuant aims to streamline the analysis process, making it accessible to researchers and bioinformaticians alike.
