## 1. Analyze — Pool Exploration & LP Position Analysis

### Pool Explorer
- Browse 30+ LP pools across multiple DEXs on Hyperliquid
- Compare TVL, APR, and fee tiers at a glance

### Position Dashboard
- Account-based LP position status and real-time profit tracking
- Asset inflow route and operation history analysis

### Curation Service
- DEX / Pool / Range optimization recommendations
- DeFi (Lending & Borrow) opportunity curation

---

## 2. Execute — LP Minting & DeFi Automation

### Seamless Minting
- One-stop bridge + swap + mint with unified UX
- Flexible price range presets (Narrow / Common / Wide / Full)

### Zap Mint
- Auto-swap and deposit all tokens for maximum LP with a single asset
- No need to manually balance token ratios

### Automation Agent
- Automated asset and position management
- Rebalancing and re-entry automation

## 3. Modular — LP Position Modular Features

### Collateral Loan
- Loans using LP positions as collateral
