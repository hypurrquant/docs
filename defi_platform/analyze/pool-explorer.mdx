## Pool Explorer

The Pool Explorer is the main view of the HypurrQuant DeFi Platform. It displays all available LP pools across multiple DEXs on Hyperliquid.

![Pool Explorer](/images/defi_platform/pool-list.png)

### Pool List

The pool list table shows the following information for each pool:

| Column | Description |
|--------|-------------|
| **Chain** | The blockchain network (currently Hyperliquid) |
| **Pair** | Token pair and fee tier (e.g., WHYPE/USDC 0.05%) |
| **DEX** | The decentralized exchange hosting the pool |
| **TVL** | Total Value Locked in the pool |
| **APR** | Annual Percentage Rate (estimated yield) |

### Search & Filter

- **Token Search** — Use the search bar to filter pools by token symbol
- **Favorites** — Click the star icon to bookmark pools for quick access

### Selecting a Pool

Click on any pool row to open the **LP Minting Panel** on the right side. The panel shows:

- Current price of the token pair
- Price range chart with liquidity distribution
- Deposit and minting interface
