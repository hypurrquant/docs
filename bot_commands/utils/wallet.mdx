<CardGroup cols={1}>
  <Card title="Manage Your Wallet" img="/images/utils/wallet_setting.jpeg">
    Create, import, export, or delete your wallet directly in Telegram using simple commands. Use **/wallet** to access all wallet-related functions.
  </Card>
</CardGroup>

- Create Wallet: Generate a new EVM-compatible wallet instantly.
- Delete Wallet: Permanently remove the current wallet from the bot.
- Import Wallet: Load an existing wallet using your private key.
- Export Wallet: Securely retrieve your wallet’s private key.
- Change Wallet: Switch between wallets you've created or imported.
- Send USDC: Transfer USDC to another address directly from the bot on Hyperliquid L1. Do not use this for CEX withdrawals.
